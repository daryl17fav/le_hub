# Icons

This directory contains SVG icons themed with The Hub's brand colors (Purple & Orange).

## Guidelines
- Use SVG format for scalability and small file size
- Apply brand colors: #6200EE (purple) and #FF9100 (orange)
- Keep icons simple and high-contrast for accessibility

## Example Structure
```
/icons
  ├── logo.svg
  ├── backpack.svg
  ├── smartphone.svg
  └── ...
```
