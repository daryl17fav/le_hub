# Audio Files

This directory contains compressed MP3 audio files for lessons.

## Guidelines
- Keep files under 500KB for low-data optimization
- Use descriptive filenames: `lesson-{id}-{stage}.mp3`
- Compress using 64kbps or lower for rural connectivity

## Example Structure
```
/audio
  ├── lesson-1-intro.mp3
  ├── lesson-1-content.mp3
  ├── lesson-2-intro.mp3
  └── ...
```
